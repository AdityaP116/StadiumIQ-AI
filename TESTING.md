# Testing Instructions

We use **Jest** for backend integration tests and **Vitest** for frontend utility tests.

## Running Backend Tests
1. Navigate to the backend directory:
   ```bash
   cd backend
   ```
2. Run the testing suite:
   ```bash
   npm test
   ```
3. To view full code coverage statistics:
   ```bash
   npx jest --coverage
   ```

## Running Frontend Tests
1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```
2. Run tests:
   ```bash
   npm test
   ```
